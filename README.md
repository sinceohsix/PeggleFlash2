# PeggleFlash

The Adobe Flash version of Peggle running in Ruffle.

[![Play Peggle](https://img.shields.io/badge/Play%20Peggle-orange)](https://sinceohsix.github.io/PeggleFlash2)

This site features a custom loader that lets you override level data and backgrounds, allowing you to load custom levels.

Level backgrounds are 440x378 `.jpg` images. Level data files are XML formatted files compressed into a .zip archive.
